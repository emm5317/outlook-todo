# Task Dashboard

## Open Tasks
```tasks
not done
sort by due
group by tags
short mode
```

## Due Within 7 Days
```tasks
not done
due before in 7 days
sort by due
```

## Follow-ups
```tasks
not done
tags include #follow-up
sort by created reverse
```

## Completed
```tasks
done
sort by done reverse
limit 20
```
